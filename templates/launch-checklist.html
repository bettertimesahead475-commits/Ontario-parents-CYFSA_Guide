Launch Checklist (Button-by-button)

GitHub

  1. Create repo: cyfsa-parent-platform
  2. Upload files (or push via git)
  3. Confirm pages load in GitHub file viewer

Netlify

  1. Netlify → Add new site → Import from Git
  2. Select your GitHub repo
  3. Build command: echo "static"
  4. Publish directory: /
  5. Deploy
  6. Open site URL → verify: Home, Law Firms, Subscriber Area, Payment Instructions

Compliance

  1. Confirm “Educational only / Not legal advice / Ontario only” shows on every page
  2. Remove any unverified lawyer listings
  3. Link only to official resources (LAO/LSO/LSRS) unless verified

First 10 subscribers (manual process)

  1. Post Facebook template #1
  2. Reply to DMs with: payment instructions + disclaimer
  3. After payment: send password + PDF downloads link
  4. Track in a spreadsheet (name, email, city, tier, paid date)
